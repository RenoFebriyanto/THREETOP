Folder for per-product icons.

Usage:
- Place product-specific icon files in the corresponding game folder.
- File name convention: either the SKU (buyer_sku_code) or a generic name like `gem.png`.
- Example path: `/public/icons/products/mobile-legends/sku_12345.png` or `/public/icons/products/mobile-legends/gem.png`.
- The app will fall back to the game image when a product-specific icon is not found.

Recommended sizes: 48x48 or 64x64 PNG/SVG for best results.
